PHYS 395 Project - Understanding the Upwelling near the Galapagos Island

Mentor: Dr. Ken Zhao

This project is designated to investigate the upwelling near the Galapagos with the usage of Oceananigans. It will include a possibility of four julia files that will output:
1. A Gaussian bathymetry for the Galapagos Island
2. The real bathymetry for the Galapagos Island
3. No bathymetry
4. An off/on clean file to turn on between the above three.

The file "galap.nc" references to the data from NOAA of the real bathymetry for the Galapagos Island.
